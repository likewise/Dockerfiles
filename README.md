# Dockerfiles

A collection of Dockerfiles for easy use of various free & open-source tools. Have fun!

## symbiyosys.Dockerfile

**Included tools:**

* Yosys
* SymbiYosys
* Z3
* Yices2
* CVC4
* Boolector
* Super Prove

## ghdl-formal.Dockerfile

Based on *symbiyosys.Dockerfile*

**Included tools:**

* All of *symbiyosys.Dockerfile*
* GHDL
* ghdl-yosys-plugin
